
# 一、开发CLI工具

## 任务一：将CGBoss CLI化，自动化测试

Playwright：CLI+Skills 实现自动化测试

```powershell
npm install -g @playwright/cli@latest

## headed 无头浏览器和有头浏览器差别
playwright-cli open google.com --headed

## cookie、登录态写入磁盘
playwright-cli open google.com --persistent

## 安装skills
playwright-cli install --skills

```


创建一个Skill，将刚刚你的操作，打开网页、操作部署、查看部署任务状态的全过程，遇到的坑都提炼出来保存到这个skill里面。后面我只要让你操作CGBoss部署，你就能调用这个skill完成任务


你把刚才所有的playWright CLI命令汇总成一个脚本，执行脚本就能获取商品前100条评论，并且保存到一个CSV文件里面。注意每一步都有合理的延时与等待，确保任务成功。脚本写完你先测试一轮。

阅读代码，把从注册开始的主体流程写一个中文的测试文档，只测试主流程即可。然后用 playwright-cli open --headed --persistent 打开网页，根据你的测试用例，完成测试
